# 🍽️ Restaurant Analytics & Rating Prediction System

An end-to-end Data Science project built for the Cognifyz Internship that explores restaurant trends, customer preferences, and rating prediction using Machine Learning.

From preprocessing and geospatial insights to a custom Hash Table with collision handling and an interactive Streamlit dashboard — this project blends Data Science with DSA to uncover what truly drives restaurant success.

Built with Python, Scikit-learn, Plotly, Folium, and Streamlit.
