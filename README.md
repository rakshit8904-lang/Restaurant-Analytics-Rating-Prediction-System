## 🚀 Setup & Run

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Add dataset
Place `Dataset .csv` (Zomato dataset) inside the `dataset/` folder.

### 3. Run Level 1 notebook
```bash
cd notebooks
jupyter notebook Level1.ipynb
```
> Run all cells. This generates `cleaned_dataset.csv` and saves map HTML files.

### 4. Run Level 3 notebook
```bash
jupyter notebook Level3.ipynb
```
> Run all cells. This trains Random Forest and saves `models/random_forest.pkl`.

### 5. Launch the dashboard
```bash
cd dashboard
streamlit run app.py
```

## 🧮 Custom Hash Table (DSA)

- **Algorithm**: Polynomial rolling hash (base-31)
- **Collision Handling**: Separate Chaining (linked list per bucket)
- **Operations**: `insert`, `search`, `delete`, `increment`, `top_n`
- **Used for**: city frequencies, cuisine frequencies, country frequencies

Run the demo:
```bash
python utils/hash_table.py
```

## 🤖 Model Selection

| Model | MAE | RMSE | R² | CV R² |
|-------|-----|------|----|-------|
| Linear Regression | ~0.55 | ~0.72 | ~0.48 | ~0.46 |
| Decision Tree | ~0.38 | ~0.52 | ~0.72 | ~0.68 |
| **Random Forest** | **~0.31** | **~0.43** | **~0.82** | **~0.80** |

**Random Forest** selected as primary model because:
1. Highest R² and lowest RMSE across train/test and K-Fold CV
2. Handles non-linear interactions (votes × cost × price_range)
3. Bootstrap aggregation provides robustness to outliers
4. Built-in feature importance for explainability
5. No feature scaling required
